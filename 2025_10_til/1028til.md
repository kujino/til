### Rubyアルゴリズム問題
- 複数の実数の出力
```ruby
Q = gets.to_i

Q.times do
# 実数や小数点の定義はループの中で定義する
  N, M = gets.split # NとMを配列に(この時点では文字列)
  N = N.to_f
  M = M.to_i #ここで小数点となるモノとそうでないものの定義

  puts format("%.#{M}f", N.round(M)) #formatで少数点以下の桁指定、0埋めは自動でされる
end
```
入力の形をまず作る、そして出力の形に整える。それが一番大事

- 数値を半角スペース埋めで出力
```ruby
N = gets.to_i

printf("%3d", N) #Nを合わせて3桁になるまで空白埋めで出力
```

- 数値をゼロ埋めで出力
```ruby
printf("%03d", N) #Nを含め3桁になるまで0埋めで出力
```

- 複数の数値を半角スペース埋めで出力
```ruby
N = gets.to_i

N.times do
    M = gets.to_i #Mの中の数字は動くのでループの中で定義
    printf("%3d\n", M) #Nが三桁になるよう空白埋め、`\n`で改行出力
end
```
