- 九九表を罫線入りで出力
```ruby
for i in 1..9 do
    for j in 1..9 do
        if j == 9 #列の値が9の場合
            printf("%3d\n", i * j)
        elsif j == 1 #行の値が1の場合
            printf " #{i * j} |"
        else #それ以外
            printf("%3d\s|", i * j)
        end
    end
    if i == 9 行の値が9なら罫線非表示
        puts nil
    else #それ以外なら罫線表示
    puts ("=" * 42 + "\n")
end
end
```

- ペアの数値の入った表を罫線入りで出力
```ruby
H, W, A, B = gets.chomp.split.map(&:to_i)

H.times do |i| #行のループ指定
    arr = [] #空配列の作成
    W.times do #列のループ指定
        arr << "(#{A}, #{B})" #W回(A, B)のペアを出力 → 配列へ
    end
    text = arr.join(" | ") #配列の要素の間に` | `を挿入
    puts text #textという値に入れておくことで要素分の罫線表示が可能に
    if i == H - 1 #最終ループは罫線非表示 0スタートなので`-1`
        puts nil
    else
         puts ("=" * text.length + "\n") #罫線の表示 textの長さ分だけ表示
     end
end

=========== 結果 ==========

(7, 8) | (7, 8) | (7, 8)
========================
(7, 8) | (7, 8) | (7, 8)
```

配列野中に`* W`で格納と考えていたけど、それでは上手くいかず  
2重ループでて配列をW回出力するといいとAIに教えていただいた  

文字数分罫線を出力という部分も配列を一つの値として入れて`length`を使えばいいと聞き、そのロジックにたどり着けなかった己の不甲斐なさを嘆いた
