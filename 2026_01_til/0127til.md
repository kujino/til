### MariaDBのインストール・起動
~ 205p

サーバー設定のところで、書籍のままの記述だとエラーになり起動が上手くいかなかった  

```bash
# /etc/my.cnf.d/mariadb-server.cnf

character-set-server-utf8 # ダメだった
character-set-server = utf8mb4 # これでいけた
```

### Mariadbの初期設定

```bash
sudo mysql_secure_installation
```
を実行し、Mriadbのrootユーザーのパスワードを設定

https://qiita.com/free-honda/items/c97a078d12bbdbbacada

### wordpressのインストール
